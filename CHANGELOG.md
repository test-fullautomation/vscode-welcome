## [0.0.2] - 2025-11-30

- Redesigned the welcome page with a new look.
- Added functionality to load URLs or local HTML files into the webview.
- Added features to open files or folders from the welcome page.
- Included library documentation for RobotFramework AIO keywords.
- Added a "Welcome" button to the status bar (bottom-right), enabling quick and seamless access to the redesigned welcome page at any time.
