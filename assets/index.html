    Browse all published packages on PyPI

  • RobotFramework Testsuites Management

    Keywords for enabling the definition of dynamic configuration values that are applied conditionally during test execution based on user defined rules.

  • RobotFramework UDS

    Keywords for sending UDS (Unified Diagnostic Services) requests and interpreting response.

  • RobotFramework DoIP

    Keywords for interacting with vehicle ECUs via Diagnostics over Internet Protocol (DoIP) to enable automated testing and diagnostics.

  • RobotFramework Extensions Collection

    Keywords for string operations, path normalization, and data pretty-printing.

  • RobotFramework Qconnection Base

    Keywords for handling continuous trace log reception and bidirectional data exchange over connections such as TCP, SSH, or serial.

  • RobotFramework Qconnection Win App

    Keywords for automating Windows applications through WinAppDriver, providing element discovery, action execution, and GUI control interaction capabilities.

  • RobotFramework Dbus

    Keywords for enabling automated testing of DBus services, allowing methods and signals to be tested both synchronously and asynchronously.

  • RobotFramework Prometheus

    Keywords for enabling Robot Framework to interact with the Prometheus monitoring system, allowing test data to be monitored and analyzed.
