Features

All-In-One package

An All-In-One package which contains all required packages, tools and libraries in one installer file:

  • Python Interpreter - version 3.9
  • Robot Framework core comes in two versions:
    Name Status Additional Info
    extended fork of version 6.1 extended version (details below)
    original 7.1 original version
  • VSCodium Editor with pre-installed helpful extensions and pre-configured workspace (details below)
  • Additional Python and Robot Framework libraries
    - maintained by RobotFramework AIO team
    - general testing related OSS libraries (last page) e.g. Appium-Python-Client and Appium, matplotlib, scipy, ...

Robot Framework core extensions

We are enhancing the Robot Framework core to meet our specific requirements, with the ultimate goal of contributing these improvements to the community version, enabling everyone to benefit from a better Robot Framework core:

  • Threading (page 13, chapter 7)
    This feature is in stabilization. A discussion how to implement for the community version you find here.
  • Loglevel "USER" (page 12, chapter 6)
    This feature is in stabilization, too. A discussion how to implement for the community version you find here.

VSCodium Editor

VSCodium Editor with pre-installed extensions, pre-configured workspace and the tutorial folder:

  • Pre-installed Python, RobotCode plugin and other useful extensions
  • Pre-configured workspace and extensions so that you just click Run/Debug button to execute Robot testcase(s)
  • "Hello World" example for immediate execution
  • The tutorial test folder for you to practice with Robot Framework

RobotFramework_TestsuitesManagement

RobotFramework AIO provides scalable management for an unlimited number of projects and variants using JSONP. JSONP, an enhanced version of JSON, enables comments, imports, and variables directly at the JSON level. This added flexibility makes it a powerful tool for efficiently managing your test suites.

  • Scalable Management: Easily manage numerous projects and variants
  • Advanced JSONP: Utilize comments, (recursive) imports, and variables directly in JSON
  • Flexible Configuration: Adapt and configure your test suites effortlessly

TestResult WebApp

RobotFramework AIO provides the solution to import the Robot Framework result file(s) (output.xml), PyTest result file(s) (as JUnitXML format) to different Test Management applications:

  • IBM Engineering Test Management: (from version 7.0.0) which is as know as Rational Quality Manager - RQM (for the older versions).
    • RobotLog2RQM tool
  • TestResultWebApp: The web application allows for sophisticated processing and visualization of test result data. It offers a comprehensive set of features including an intuitive dashboard for tracking test execution metrics, detailed datatables for reviewing individual test outcomes, and runtime optimization views. The Diff view enables easy comparison of test results across multiple runs, enhancing analysis and decision-making.
    With flexible filtering, commenting, and collaboration tools, this platform supports effective test result management and comparison, making it an essential tool for teams seeking in-depth insights into their testing process.
    • RobotLog2DB tool
    • PyTestLog2DB tool
