\hypertarget{description-getting-started}{%
\section{Getting Started}\label{description-getting-started}}

\textbf{VSCodeWelcome} is a VSCodium extension that enhances the user
experience by displaying a RobotFramework-AIO introduction for new
users.

\hypertarget{description-usage}{%
\section{Usage}\label{description-usage}}

\subsection{How to Install}

To install the \textbf{Robotframework Welcome} extension:

\begin{enumerate}
    \item Download the extension package or clone the repository.
    \item Run the build script \texttt{build\_vsix.sh} to generate the VSIX file.
    \item Open Visual Studio Code or VSCodium.
    \item Select the generated \texttt{robotframework-welcome-x.x.x.vsix} file to install the extension.
\end{enumerate}

\subsection{How to Set a Custom URL for the Welcome Page}

To customize the URL displayed on the welcome page:

\begin{enumerate}
    \item Look for the setting named \texttt{robotframeworkWelcome.welcomeUrl} in the \texttt{package.json} file.
    \item Enter the desired URL (e.g., \url{https://your-custom-url.com}) or a local file path (e.g., \texttt{file:///absolute/path/to/file.html}).
    \item Save the changes to the \texttt{package.json} file.
    \item Run the script \texttt{build\_vsix.sh} to build the extension.
    \item Install the generated \texttt{robotframework-welcome-x.x.x.vsix} file in Visual Studio Code or VSCodium.
\end{enumerate}

The next time the welcome page is displayed, it will load the custom URL or local file you specified.
